int trie_autocomplete ( TrieNode *  root,
const char *  prefix,
char **  results,
int  max_results,
int  max_word_len 
)

Collect all words that begin with prefix into a caller-supplied array.

Memory ownership: each written element of results is a heap-allocated string that the caller must free(). On error, any partially allocated strings are freed and the array is left in an unspecified state.

Parameters
rootRoot of the trie.
prefixNUL-terminated prefix to search for.
resultsOutput array of at least max_results char* pointers.
max_resultsMaximum results to return; must be > 0.
max_word_lenMaximum word length to include; must be > 0.
Returns
Number of results written on success, 0 if none, or -1 on invalid args.
